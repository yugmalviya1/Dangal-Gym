# ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/yugmalviya1/Dangal-Gym.git
```

Move into the project directory:

```bash
cd Dangal-Gym
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
